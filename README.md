# Q&A Bot

Q&A Bot is a simple, interactive chatbot built with React and Framer Motion. This bot provides users with pre-defined options to ask questions and receive responses, simulating a conversation.

You can easily integrate this into your website and create an list of few questiongs with answers

## Features

- **Animated Chat Interface**: Smooth animations for messages using Framer Motion.
- **Pre-defined Responses**: Users can select from a set of pre-defined options to interact with the bot.
- **Responsive Design**: The chat interface is responsive and works well on various screen sizes.

### Customization
You can easily customize the questions and responses by modifying the allOptions array within the App.tsx file.
